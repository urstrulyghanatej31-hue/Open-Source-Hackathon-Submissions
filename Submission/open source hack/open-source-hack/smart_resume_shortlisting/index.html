Overview

Data-driven hiring with less manual screening

Recruiters often receive hundreds of resumes for a single job opening. Manual review is slow, inconsistent, and can introduce unconscious bias. This system uses Machine Learning and Natural Language Processing to parse resumes, evaluate relevance, and rank candidates for a specific job role.

Key Features

Built for candidates, recruiters, and AI scoring

Candidate Features

  • Upload resumes in PDF or DOCX format
  • View application status
  • Manage candidate profile details

Recruiter Features

  • Create and manage job descriptions
  • Upload multiple resumes
  • Filter, rank, and download shortlists

AI/ML Features

  • Resume parsing and skill extraction
  • Keyword and experience analysis
  • Resume-job similarity scoring

System Architecture

From resume upload to ranked shortlist

Recruiter uploads job description
Candidates upload resumes
Parser extracts skills, education, experience, and certifications
NLP engine preprocesses resume text
ML model calculates matching score
Dashboard displays ranked candidates
